\# STFL



Simple Text Format Language



STFL is an AI-friendly web language designed to be simpler than HTML and easier to generate with AI.



\## Features



\- TXT editable

\- AI friendly

\- HTML compiler

\- Simple syntax

\- Indentation-based structure
